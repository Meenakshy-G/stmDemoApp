## stmDemoApp 
STM32 Devkit familiarisation - GPIO and UART. \
Blinking led using gpio control and using UART printing statement into the 
terminal. 

# GPIO Control 
GPIO pin PB14 used for user LED (RED).

# UART
UART3 used for printing a statement into the terminal.

# Software Used
STM32CubeIDE 1.18.1 is used for building the project.
